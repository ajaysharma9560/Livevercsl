# 📱 App Status Monitor

Real-time Android app online/offline status monitor using Vercel.

## 🚀 Features

- ✅ Real-time status updates (Active/Offline)
- 🟢 Green dot when app is open
- 🔴 Red dot when app is closed
- 📜 Status history tracking
- 🔄 Auto-refresh every 5 seconds

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Vercel Serverless Functions
- **Android**: Kotlin + Retrofit/OkHttp
